## âš ï¸ **Common Pitfalls**

### **1. Complexity for Complexity's Sake**
**Problem:** Using advanced techniques without purpose.
**Solution:** Every advanced choice must serve story/theme/character.

### **2. Losing Reader Trust**
**Problem:** Too many tricks confuse or frustrate readers.
**Solution:** Be clear about your narrative contract early.

### **3. Unearned Sophistication**
**Problem:** Advanced techniques before mastering basics.
**Solution:** Nail fundamentals first, then experiment.

**Problem:** Structural complexity overshadows characters.
**Solution:** Character always comes first, structure serves character.
