## ğŸ“– **Source Tracking**

### **Why Track Sources?**

- Verify facts later
- Give credit if required
- Return to source for more info
- Evaluate reliability
- Share with beta readers/editors
- Defend accuracy if questioned

---

### **Citation Format (Simple Version):**

**Books:**
```
Author Last Name, First Name. Book Title. Publisher, Year.
Example: Smith, John. Victorian London. Oxford Press, 2015.
```

**Articles:**
```
Author. "Article Title." Publication, Date. URL
Example: Johnson, Mary. "Police Procedures in 1889." History Today, Jan 2020.
```

**Interviews:**
```
Interviewee Name. Personal interview/Email interview. Date.
Example: Williams, Sarah (Detective). Personal interview. March 15, 2024.
```

**Websites:**
```
"Page Title." Website Name. URL. Access Date.
Example: "Victorian Fashion." Victoria & Albert Museum. [URL]. Accessed March 2024.
```

---

### **Source Reliability Rating:**

Add reliability notes to sources:

â­â­â­ **Primary/Expert:** Firsthand account, academic expert, official records
â­â­ **Secondary/Reliable:** Well-researched book, reputable publication
â­ **Tertiary/Verify:** Wikipedia, blog, need to cross-check

## âœ… **Research Workflow**

### **While Researching:**

1. **Capture:** Take notes in chosen system
2. **Cite:** Record source immediately
3. **Tag:** Add relevant tags/categories
4. **Reflect:** Note why this matters to story

### **Transferring to Research Bible:**

**Weekly/Monthly:**
- Review all notes
- Add key facts to research bible
- Discard irrelevant information
- Consolidate duplicates

### **While Writing:**

**Keep Open:**
- Research bible
- Current chapter research
- Key source documents

**Mark Gaps:**
```
She walked through the [RESEARCH: Victorian train station description] 
and approached the [RESEARCH: what was ticketing process?].
```

**Update As You Go:**
- Note new questions that arise
- Flag facts you use (so you remember source)
