- Literary fiction embraces more devices
- Commercial fiction uses them sparingly

## ğŸ¯ **Practice Exercises**

### **Exercise 1: Metaphor Hunt**
- Read a chapter of your favorite novel
- Identify 10 metaphors or similes
- Analyze what makes them effective
- Write your own variations

### **Exercise 2: Symbol Creation**
- Choose an everyday object
- Write a scene where it has symbolic meaning
- Don't explicitly state the symbolism
- Test it on readers

### **Exercise 3: Foreshadowing Retrofit**
- Take a completed short story
- Add three instances of foreshadowing
- Make them subtle but discoverable
- Check if the ending feels inevitable

### **Exercise 4: Sensory Imagery**
- Describe a location using all five senses
- Use no visual descriptions
- Create vivid imagery through sound, smell, touch, taste
- Add visual details back strategically
